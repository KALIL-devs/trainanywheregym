# Train Anywhere Gym - Modern Web App Portfolio

A production-ready, highly interactive landing page built to demonstrate my ability to translate complex design requirements into clean, performant, and maintainable code. 

As a fresher, I built this project to showcase that I can do **actual work**—going beyond simple tutorials to build a fully responsive, modern web application with advanced animations and a scalable component architecture.

## 🎯 Purpose

This project is a functional frontend application tailored for an elite fitness coach. It demonstrates my proficiency in building real-world products using the modern React ecosystem, focusing on UI/UX, performance, and responsive web design.

## 🚀 Technical Highlights & Skills Demonstrated

- **Modern React & Next.js App Router:** Leveraged the latest Next.js 15+ features, demonstrating an understanding of server vs. client components, routing, and modern React patterns.
- **Advanced UI/UX Implementation:** Translated a premium dark-mode aesthetic into code, utilizing complex CSS techniques like glassmorphism, background blurs, and gradient overlays.
- **Complex Animations:** Used **Framer Motion** to implement scroll-triggered parallax effects, staggering reveals, and interactive micro-animations without compromising performance.
- **Scalable Styling with Tailwind CSS v4:** Wrote clean, utility-first CSS using the latest Tailwind version, avoiding messy stylesheets and ensuring strict design consistency.
- **Component-Driven Architecture:** Organized the codebase into modular, reusable components (`Hero`, `About`, `Programs`, `Philosophy`, etc.) to ensure the code is maintainable and scalable.
- **Responsive Web Design:** Ensured a seamless user experience across all devices (mobile, tablet, desktop) using fluid typography and CSS Grid/Flexbox layouts.

## 🛠️ Tech Stack

- **Framework:** [Next.js](https://nextjs.org/) (React)
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
- **Icons:** [Lucide React](https://lucide.dev/)
- **Language:** TypeScript

## 🧠 What I Learned

While building this project, I deepened my understanding of:

## 💻 Running the Project Locally

Clone the repository and install the dependencies:

```bash
npm install
# or
yarn install
# or
pnpm install
```

Run the development server:

```bash
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the live application.
